# Dayflow Support

This document is the source copy for the public Support URL required in App Store Connect.

## English

For Dayflow support, email: [Exitze@icloud.com](mailto:Exitze@icloud.com)

Please include:

- App name: Dayflow
- iOS version
- iPhone model
- A short description of the issue
- The screen where the issue happened: Home, Calendar, Statistics, or Settings

Dayflow stores data locally on your device. Data deleted through app reset or app uninstall may not be recoverable from a server because the current version does not use server-side storage.

## Русский

Поддержка Dayflow: [Exitze@icloud.com](mailto:Exitze@icloud.com)

Пожалуйста, укажите:

- Название приложения: Dayflow
- Версию iOS
- Модель iPhone
- Короткое описание проблемы
- Экран, где появилась проблема: Главная, Календарь, Статистика или Настройки

Dayflow хранит данные локально на устройстве. Данные, удаленные через сброс приложения или удаление приложения, могут быть недоступны для восстановления с сервера, потому текущая версия не использует серверное хранение.
