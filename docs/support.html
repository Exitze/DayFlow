Support

Need help with Dayflow?

For support, include the app name, iOS version, iPhone model, and a short description of the issue.

Contact: Exitze@icloud.com

English

Email support: Exitze@icloud.com

Dayflow stores data locally on your device. Data deleted through app reset or app uninstall may not be recoverable from a server because the current version does not use server-side storage.

Before contacting support, please check whether the activity, note, or shift belongs to the selected calendar day.

If reminders do not appear, check iOS Settings → Notifications → Dayflow and make sure notifications are allowed.

Русский

Поддержка по email: Exitze@icloud.com

Dayflow хранит данные локально на устройстве. Данные, удаленные через сброс приложения или удаление приложения, могут быть недоступны для восстановления с сервера, потому текущая версия не использует серверное хранение.

Перед обращением проверьте, что активность, заметка или смена относятся к выбранному дню календаря.

Если напоминания не приходят, откройте настройки iOS → Notifications → Dayflow и убедитесь, что уведомления разрешены.
